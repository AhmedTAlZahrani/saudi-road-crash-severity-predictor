# Saudi Road Crash Severity Predictor

![Tests](https://github.com/AhmedTAlZahrani/saudi-road-crash-severity-predictor/actions/workflows/test.yml/badge.svg)

ML pipeline for predicting road crash severity on Saudi Arabia's highway network, supporting Vision 2030 road safety objectives.

---

## What It Does

Classifies crash outcomes into four severity levels -- property-only, minor injury, severe injury, and fatal -- using Saudi-specific risk features like sandstorm visibility, extreme heat tire blowout risk, Saher speed camera zones, and camel crossing hazards on rural intercity highways.

## Model Results

| Model | Accuracy | Precision | Recall | F1 (weighted) |
|-------|----------|-----------|--------|----------------|
| Logistic Regression | 72.8% | 71.4% | 72.8% | 71.2% |
| Random Forest | 80.5% | 79.8% | 80.5% | 79.9% |
| **XGBoost** | **82.3%** | **81.7%** | **82.3%** | **81.9%** |
| LightGBM | 81.9% | 81.2% | 81.9% | 81.5% |

Best model: XGBoost with SMOTE oversampling (weighted F1 = 0.819, stratified 5-fold CV).

### Key Features by SHAP Importance

1. `estimated_speed` -- impact speed is the strongest severity predictor
2. `seatbelt` -- no seatbelt dramatically increases injury severity
3. `speed_visibility_risk` -- compound risk of high speed in low visibility
4. `visibility_km` -- sandstorms and fog drive severe/fatal outcomes
5. `num_vehicles` -- multi-vehicle crashes escalate severity

## Usage

### Generate Data and Train

```python
from src.simulate_crashes import generate_crash_data
from src.data_loader import load_crash_data
from src.transforms import build_crash_features, build_preprocessor
from src.train import train_and_compare

# Generate synthetic crash data
df = generate_crash_data(n_records=50000)
df.to_csv("data/saudi_crashes.csv", index=False)

# Load, transform, and compare models
X, y = load_crash_data("data/saudi_crashes.csv")
X = build_crash_features(X)
preprocessor = build_preprocessor(X)
X_processed = preprocessor.transform(X)

results = train_and_compare(X_processed, y)
print(results)
```

## Dataset

Synthetic data generated by `src/simulate_crashes.py` (50,000 crash records) calibrated to Saudi General Authority for Statistics transport reports. Features include:

- **Sandstorm visibility** (0.1-2 km during dust events)
- **Extreme heat** (45C+ tire blowout risk on highways)
- **Saher zones** (speed camera enforcement areas)
- **Prayer time effects** (reduced traffic density, higher individual speeds)
- **Camel crossing risk** (rural intercity highways at night)
- **Ramadan / Hajj** period flags
